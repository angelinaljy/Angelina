# Crypto.com
## 关于crypto.com

我们是开创性的加密货币支付平台，我们旨在为客户提供比银行更广泛灵活的金融服务。主要产品包手机钱包App，Visa卡，crypto链以及交易所平台。
总部位于香港，在马耳他和保加利亚设有办公室,另有深圳研发中心设立于南山高新园。我们的主要市场是在欧洲和北美，以及亚洲的新加坡等众多发达国家和地区。技术团队都是来自国际各大科技公司的精英人才，例如：IBM，腾讯，Facebook，Microsoft等IT公司。
